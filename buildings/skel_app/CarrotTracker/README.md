# 🥕 CarrotTracker - Farm Lore
A system to monitor the growth of ginger carrots in the north field.

Gingilla, the ginger chinchilla, scurries around the new CarrotTracker. 
Every brick is a microservice, and every wire is a route in the big stack.

---
## Technical Specification
- **Backend:** FastAPI (Python 3.11)
- **Frontend:** React + Vite + Tailwind
- **Database:** postgres
- **Port:** 8000

## How to Run
1. Ensure Docker is running.
2. Run `docker-compose up --build`.
